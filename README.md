# SENTIMENT-AND-ENGAGEMENT-INSIGHT
Sentiment and engagement insights analysis using Python and Power Bi

Data Cleaning and Exploratory Data Analysis was done and an interactive dashboard for visualization was created using Power Bi

This dashboard delivers insights into customer sentiment, engagement patterns, and platform performance, enabling businesses to optimize communication strategies and enhance customer experience

**Key Insights**

Positive sentiment dominant the dashboard which is a strong signal of customer satisfaction
Highest activities comes from USA, UK, Canada and Australia
Instagram and Twitter show higher engagement rates compared to facebook. This implies that market focus should be shifted to high performing platforms.
